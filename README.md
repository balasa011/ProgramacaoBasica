# Conteúdo Inicial
Este é um repositório de estudo-aprendizagem, será publicado aqui a solução de problemas diversos utilizando Java para consolidar o aprendizado de algorítimos e lógicas de programação.
## Problemas
Os problemas podem ser encontrados no repositório \problemas. Estão dividios em diferentes etapas do processo de aprendizagem de programação. Sendo eles:

### Básico
Estes são problemas simples que exigem apenas conhecimento de operadores lógicos, operadores aritméticos e condicionais.
### Loops
Estes são problemas que necessitam algum tipo de loop ou iteração sobre algum elemento. Esses problemas não exigem o uso de vetores ou objetos.
### Vetores
Estes são problemas que com o uso de vetores se tornam triviais.
### Projetos Finais
Estes são problemas que exigem o uso de alguma estrutura de dados. A implementação de novas classes, funções ou otimização é fundamental.
## Soluções
As soluções seram feitas em Java, cada exercicio será feito em uma classe própria com execução independente - método main() proprio.
